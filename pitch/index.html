How It Works

Data → Score → Recommendation

01

Ingest

3,224 mymizu stations + Tokyo 23-ward GeoJSON + Ministry of Environment waste data

02

Spatial Join

@turf/turf point-in-polygon → station count per ward → coverage density (stations/km²)

03

Score

severity = waste_per_capita / (stations/km² + 0.01) → normalize [0, 100]

04

Find Gaps

40×40 candidate grid → max distance to nearest station → top 3, ≥350m apart

05

Generate Brief

Claude Sonnet synthesizes ward data + gap locations → 3-sentence intervention plan
